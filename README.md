# .vim

### Настройки редактора `VIM` (MacOS) с русским хелпом.

1. Клонируйте папку `.vim` в домашнюю директорию вашего мак.
        
```shel
git clone 'https://github.com/ibo7/.vim.git'
```

1. Cоздайте алиас в терминале командой: `ln -s ~/.vim/vimrc ~/.vimrc`
1. Запустите Vim.
1. Нажмите два раза клавишу ` ; ` и внизу, после двоеточия введите команду
   `PlugInstall`.
1. Перезапустите `VIM`. Нажмите `Return`.
